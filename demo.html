HTML5 Cross Browser Compatible AJAX File Upload Demo

This is a demonstration of a cross browser AJAX file upload solution using XHR (HTML5). Allowed upload filetypes are configurable. You can optionally email the upload as an attachment where required.

Tested with Firefox, Chrome, IE 9,10,11 Safari - Desktop, i/Phone/Pad/Pod.

Source : https://github.com/gaiterjones/cross-browser-ajax-file-uploader

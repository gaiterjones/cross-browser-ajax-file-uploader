Cross Browser Compatible AJAX File Upload Demo

This is a demonstration of a cross browser AJAX file upload solution using XHR (HTML5) and Flash (Uploadify). Flash or HTML5 will be autodetected based on browser capability. Allowed upload filetypes are configurable. For the demo you can toggle between Flash and XHR using the toggle button. You can optionally email the upload as an attachment where required.

Tested with Firefox, Chrome, IE 9,10, Safari - Desktop, i/Phone/Pad/Pod.
